const std::vector< class Intersection > Triangle::localIntersect ( const class Ray &  localRay)
virtual

Finds all the intersections by employing the Möller–Trumbore algorithm. A ray that misses a triangle should not add any intersections to the vector of intersections. A ray that strike a trinagle should add exactly one intersection to the list.
