std::vector< Intersection > Sphere::intersect ( const class Ray &  ray)

Returns a collection of Intersections. The functions calculates discrimant value using Joachimsthal's quadratic equation and uses its value to determine the number of solutions.

Parameters
Rayray Reference to the ray to be checked whether or not it intersects with the sphere.
Returns
A vector of all solutions of the equations for discriminants bigger or equal to zero and the sphere that is intersected, otherwise an empty vector.
